## Interactive 2026 Calender
A modern, highly interactive web-based calendar for the year 2026 with a clean, premium UI and smooth animations.

# Prompt for the Calender

Create a **modern, highly interactive web-based calendar for the year 2026** with a clean, premium UI and smooth animations.

### Core Features

* A **full 2026 calendar view** (monthly and yearly navigation)
* Clicking on any date opens a **beautiful animated popup modal**
* Festival dates show **special themed popups** with festive colors, icons, and subtle confetti or glow effects
* Allow users to **add birthdays and special dates**, each with custom labels and emojis
* Highlight birthdays and special days directly on the calendar with badges or dots

### Personalization

* A **customizable title at the top** (example: “Shree’s 2026 Planner”)
* Editable subtitle or short quote under the title
* Save user inputs using local storage so data persists after refresh

### To-Do & Notes System

* A **To-Do List with exactly 5 sections**

  * Each section supports adding, editing, completing, and deleting tasks
  * Smooth checkbox animations and strike-through effects
* A **Notes panel** for free-form text
* Notes and todos should be linked to specific dates when possible

### Design & Style

* **Modern typography** (use fonts like Poppins, Inter, or Montserrat)
* Unique color palette:

  * Primary: Deep Indigo / Royal Purple
  * Accent: Neon Cyan or Soft Mint
  * Background: Light glassmorphism or soft gradient
* Use **hover animations** on:

  * Calendar dates
  * Buttons
  * Todo items
* Add subtle **micro-interactions** (scale, glow, shadow lift, ripple effects)

### Animations

* Smooth transitions using CSS animations or JavaScript
* Popup modals should fade + slide or scale in
* Hover effects should feel responsive and premium
* Optional dark mode toggle with animated transition

### Technical Expectations

* Built using **HTML, CSS, and JavaScript**
* Clean, well-structured, and commented code
* Responsive for mobile, tablet, and desktop
* No external backend required (use localStorage)

### Extra Touches

* Today’s date highlighted with a special glow
* Festival popups include icons and themed backgrounds
* Overall feel should be **minimal, elegant, and futuristic**

The final result should feel like a **premium personal digital planner for 2026**, visually impressive, smooth to use, and highly customizable.
